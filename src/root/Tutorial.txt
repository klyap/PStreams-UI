Tutorial
########################################
Create a graph
========================================
Go to flowhub.com and create a graph

Download the JSON file by hitting "save"

Execute graph
========================================
Navigate to the directory of this tool on terminal/command line.

Run the tool.

Ie. If you named your graph "mygraph" in the Flowhub UI, type:

``python run.py graphname.json``

You will see an animation of your graph with data!
